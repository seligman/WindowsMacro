Macro Definitions

Each key is represented by one or more characters. To specify a single keyboard character, use the character itself. For example, to represent the letter A, use "A" for the string. To represent more than one character, append each additional character to the one preceding it. To represent the letters A, B, and C, use "ABC" for string.

The plus sign (+), caret (^), percent sign (%), tilde (~), and parentheses have special meanings. To specify one of these characters, enclose it within braces ({}). For example, to specify the plus sign, use {+}. To specify brace characters, use {{} and {}}.

Special Codes

  • ACTIVEAPP
    Repositions the active window.
    LEFTRepositions the window the left half of the screen.
    MAXMaximizes the window.
    MINMinimizes the window, or restores it if maximized.
    MINOTHERMinimizes all other windows.
    MONDOWNMoves the window over a monitor.
    MONLEFTMoves the window over a monitor.
    MONRIGHTMoves the window over a monitor.
    MONUPMoves the window over a monitor.
    RIGHTRepositions the window the right half of the screen.
    STRETCHStretches the window to the top and bottom.
  • ADDVAR <value>
    Add value to the current variable. See SETVAR and TYPEVAR.
  • BEGIN
    Start point for a LOOP command.
  • BOARD
    Types the contents of the clipboard (without being parsed). This differs from pasting because the keystrokes for each letter of the text in the keyboard are simulated, and any formatting contained in the clipboard is lost.
  • CD <code>
    Controls various aspects of the CD-ROM drive.
    CLOSECloses the tray of the CD drive.
    OPENOpens the tray of the CD drive.
  • HIBERNATE
    Causes the system to enter hibernation mode.
  • LAUNCH <application>
    This launches a windows application. For instance: "{LAUNCH notepad.exe}" will start notepad.
  • LOOP <count>
    Run a loop, starting over at the previous BEGIN command.
  • PASTESCRUB
    Pastes a scrubbed version of the clipboard. This removes any formatting or other non-text elements from the clipboard and pastes the results using Control-V
  • ROTATEDISPLAY <degrees>
    Rotate the display ninety degrees, or specify one of "0d", "90d", "180d", or "270d" to rotate to a specific orientation.
  • SETVAR <value>
    Set a variable to the value specified. See ADDVAR and TYPEVAR.
  • SLEEP <milliseconds>
    Sleeps for the specified time. For instance: "{SLEEP 1000}" will cause the app to sleep for one second.
  • SUSPEND
    Causes the system to enter suspend mode.
  • SWITCHTO <window name>
    Switches to the first window with the matching string found in its window title. For instance: "{SWITCHTO notepad}" will bring the first instance of notepad to the foreground.
  • SYSCOMMAND <code>
    Sends various windows system commands to the active window.
    CLOSEClose the active window.
    HSCROLLScroll horizontally.
    MAXIMIZEMaximize the active window.
    MINIMIZEMinimize the active window.
    MOVEMove the active window.
    NEXTWINDOWMove to the next window.
    PREVWINDOWMove to the previous window.
    RESTORERestore window to normal position and size.
    SCREENSAVEExecutes the screen-saver.
    SIZESize the active window.
    TASKLISTActivate the Windows Task Manager.
    VSCROLLScroll vertically.
  • TYPEVAR
    Types the current variable as text. See ADDVAR and SETVAR.

Key Modifiers

To specify keys combined with any combination of the shift, control, and alt keys, precede the key code with one or more of the following codes:

KeyCode
Alt%
Control^
Shift+

To specify that any combination of shift, control, and alt should be held down while several other keys are pressed, enclose the code for those keys in parentheses. For example, to specify to hold down shift while E and C are pressed, use "+(EC)". To specify to hold down shift while E is pressed, followed by C without shift, use "+EC".

To specify repeating keys, use the form {key <number>}. For example, {LEFT <42>} means press the left arrow key 42 times; {h <10>} means press the letter H 10 times.
